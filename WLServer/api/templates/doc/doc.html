实现的比较粗糙,还有好多功能未实现

  1. 注册 api/register.json

    • 请求方式: POST

    • 返回结果：json

      • uid - 用户名
      • pwd - 密码(建议先进行加密,若进行了加密处理,以后所有的pwd参数都应进行同样的加密处理)

      JSON:

                                                      
      --- success ---
      {
					  "success": {
					    "message": "注册成功",
					  }
					}
			
                                                
      --- error ---
      {
					  "error": {
					    "message": "错误信息",
					  }
					}

  2. 登录 api/login.json

    • 请求方式: POST

  3. 发送一条消息 api/chat/upload.json

    • 请求方式: POST

    • 返回结果：json

      • text - 消息内容(文字消息为:消息内容; gif表情消息为:gif表情对应的图片名称;语音,图片消息为:资源的url)
      • type - 消息类型(0, "文本消息"),(1, "gif表情消息"),(2, "图片消息"),(3, "语音消息")
      • access_token - 授权标识(登录时成功时可以得到)
      • to_user - 接收者uid

      JSON:

                                                      
      --- success ---
      {
					  "message": {
					    "m_id": "消息ID",
					    "text": "消息内容",
					    "create_time": "创建时间",
					    "type": "消息类型 整数",
					    "from_user": {
					      "uid": "用户名",
					      "name": "昵称",
					      "photo": "头像",
					    },
					    "to_user": {
					        "uid": "用户名",
					        "name": "昵称",
					        "photo": "头像",
					      },
					  },
					}
			
                                                
      --- error ---
      {
					  "error": {
					    "message": "错误信息",
					  }
					}
